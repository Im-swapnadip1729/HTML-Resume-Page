# HTML-Only Resume Project

This project is a single-page resume built strictly using **HTML**. It focuses on semantic structure and traditional HTML attributes to achieve a clean layout without the use of external or internal CSS.

### The layout includes:

* **Skills**: An unordered list of technical proficiencies.
* **Experience & Education**: Data organized within bordered tables.
* **Projects**: Sectioned highlights of recent work.
* **Contact**: Bolded contact details and a footer.
